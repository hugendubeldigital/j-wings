wingS mailing lists

There are currently four mailing lists available for wingS:

  • wingS-announce - all announcements are posted into this list
  • wingS-users - if you only want to use wingS and have questions, subscribe to this mailing list
  • wingS-developers - if you want to help develop wingS, you should subscribe here
  • wingS-cvs - all commited patches to the cvs repository are posted into this list
