wingS development

We are using a cvs pserver at cvs.wings.to.com.

anonymous (read) access:

  • repository: :pserver:anonymous@cvs.wings.to.com:/wings
  • password: <none>
  • module: wings

users with write access:

  • repository: :pserver:<userid>@cvs.wings.to.com:/wings
  • password: <you_should_know>
  • module: wings

You can also use cvsweb to browse the CVS Repository

Feel free to contribute! Send patches / fixes / extensions to wings-developers@wings.to.com. Contact Michael Reinsch to get write acces to the CVS Repository.
