wingS development

We are using a cvs pserver at cvs.wings.to.com.

anonymous (read) access:

  • repository: :pserver:anonymous@cvs.wings.to.com:/wings
  • password: <none>
  • module: wings

to checkout the current release, use
  cvs -d :pserver:anonymous@cvs.wings.to.com:/wings co -r release-1_0-beta1 wings

to checkout the most current version (HEAD), use
  cvs -d :pserver:anonymous@cvs.wings.to.com:/wings co wings

You can also download the latest dayly snapshots here (those might not work) or browse in the CVS Repository.

users with write access:

  • repository: :pserver:<userid>@cvs.wings.to.com:/wings
  • password: <you_should_know>
  • module: wings

Feel free to contribute! Send patches / fixes / extensions to wings-developers@wings.to.com. Contact Michael Reinsch to get write access to the CVS Repository.
