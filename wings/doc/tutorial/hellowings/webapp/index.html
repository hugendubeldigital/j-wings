HelloWings - Tutorial for wingS

This site is the initial site to access the HelloWings Tutorial.

Within this tutorial all required steps are discussed to set up a working wingS application from scratch. Even if you are not familiar with setting up a servlet container and running applications in it, this tutorial will help you to success.

Since most professional web servers are running under an UN*X environment, I only cover here the setting up of a Linux (which is UNIX) server. It is not very difficult to do the required steps in other environments, e.g. Windooze, because anything related to the servlet container is platform independent.

Now follow me to the tutorial (the tutorial itself is also a wingS application).
